# Escola-Paulucci-Molodoy-
BiblioDesk é um sistema Java Desktop para controle de empréstimos em biblioteca escolar. Substitui planilhas por gestão digital simples. Permite cadastrar alunos e livros, registrar empréstimos e devoluções, consultar disponibilidade e atrasos. Limite de 3 livros por aluno, prazo de 7 dias e funcionamento offline com persistência simples.
